# NexoraBot Chat Server

## How to Run Locally

1. Add your OpenAI API key in a `.env` file like this:

```
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxx
```

2. Install dependencies:

```
npm install
```

3. Start the server:

```
node server.js
```

4. POST requests can be made to:

```
http://localhost:3000/api/chat
```
